# Son d'ambiance pour SUNA

Place ton fichier audio ici, nommé "ambiance.mp3".

Le bouton 🔇/🔊 contrôle le play/pause.
Le slider contrôle le volume (25% par défaut).

Pour un son d'ambiance désertique, pense à :
- Vent léger
- Bruits de sable
- Musique orientale douce
- Sons lointains de village

Format recommandé : MP3, boucle infinie.
